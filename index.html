Camera Parameters

The numbers below will update as you interact with the map. Copy the code snippet into your app to replicate the current view.

center: [-2.9622, 54.4283],
zoom: 15.16,
bearing: 17.00,
pitch: 77.00,
padding: [0, 0, 0, 0],

You may also wish to specify the following optional parameters:

maxPitch: 85, // experimental!
minZoom: 6,
maxZoom: 16,

This tool uses Ordnance Survey's Open Zoomstack and Terrain 50 data, which you can download and use for free.
